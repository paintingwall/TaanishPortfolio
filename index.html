Detailed Spotlight: Dragon ShootR

Overview: I created a simple shooting game using block-based coding in the MIT App Inventor software. I created a game where a human character moves left and right and shoots bullets to score points by hitting a flying dragon before time runs out.


Technical Skills: I used variables to keep track of the score, timer, and lives. I implemented event-driven programming so the game reacts to collisions and timer events. This project helped enhance my problem-solving skills, specifically when debugging bullet collision and timer logic.


Collaboration: My partner handled the pixel art via Piskel, while I handled all programming and logic.
